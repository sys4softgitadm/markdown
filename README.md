Este é um parágrafo normal.
Linhas seguintes continuam a fazer parte da primeira linha.

Temos que dar um espaço adicional para criar um novo parágrago.  
E pelo menos um espaço em branco no final de uma frase para fazer quebra de linha.

# Enfatização
Texto normal

texto a **negrito** ou ainda a __negrito__

texto com *itálico* ou ainda com _itálico_

texto com ***itálico negrito*** ou ainda com  ___itálico negrito___

# Código em linha e em bloco
Podemos adicionar `codigo_em_linha()` e depois mais `outro_codigo();`

---

Para código em bloco:
```json
{
    "Nome": "João",
    "Apelido": "Ribeiro"
}
```

===

ou em PHP
```php
$nome = "João";
$apelido = "Ribeiro"
```
